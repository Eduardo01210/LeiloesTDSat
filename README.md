# Sistema de Leilões

## Sobre o projeto
Este projeto consiste em um sistema de leilões desenvolvido em Java, com integração a banco de dados MySQL. O sistema permite cadastrar produtos, listar os produtos cadastrados e alterar o status de venda.

## Funcionalidades
- Cadastro de produtos
- Listagem de produtos
- Atualização do status do produto para vendido

## Tecnologias utilizadas
- Java
- MySQL
- NetBeans
- Git
- GitHub
